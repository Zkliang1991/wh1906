总结

经过这一个月的学习，让我的见识又有新的增长。 首先是node.js，使用express脚手架快速搭建一个项目。 Web 应用程序 Express 是一个保持最小规模的灵活的 Node.js Web 应用程序开发框架，为 Web 和移动应用程序提供一组强大的功能。 其次是vue，使用vue-cli也能快速的搭建一个项目。在这里我就不在详细说明它的使用方法了。 vue是我们进入三阶段以来学习最长得一个框架，从学习时间上来看，我们也能猜到他的重要性。所以在学vue千万不要松懈，一定要打好基础，要多练习，多和别人沟通。不要不懂装懂，这样害的是你自己。我个人认为vue中最难的就是父子组件之间的双向数据绑定，计算属性，和路由守卫。在做vue项目的时候，我们会使用到一些框架，例如:element-IU ,vant 在使用框架的时候一定要看清官网给的api，不能自己瞎定义一些文档中没有的属性。在学习vue的时候我还遗留了一些问题没能解决，例如：vuex的使用方法不太明确，兄弟组件之间的数据双向绑定等等。
